Тээврийн хэрэгслүүдийн хурдны массив, хурдны хязгаар гэсэн хоёр өгөгдлийг ашиглан, [Хурд хэтрүүлсэн тоо, Хэтрүүлсэн дундаж хурд] гэсэн хоёр элементийн массивыг буцаадаг функц зохионо.
