# Халдварласан

1988 оны 11-р сарын 2-нд анхны томоохон интернет өт гарч, нэг өдрийн дараа интернетэд холбогдсон компьютеруудын 10 орчим хувийг халдварлажээ. Энэхүү сорилтод танд интернетийн хорхой гарснаас хойш хэд хэдэн хоног өнгөрч байгаа бөгөөд та дараах дүрмийн дагуу хэдэн компьютер халдвар авсан болохыг тодорхойлох хэрэгтэй. 0-р өдөр эхний компьютер халдвар авсан байна. Дараагийн өдөр бүр халдвар авсан компьютерийн тоо хоёр дахин нэмэгддэг. 3 дахь өдөр тутам вирус тархсаны дараа засвар хийж, халдвар авсан компьютерийн тоог 20% бууруулдаг. Засвар хийсэн компьютерийн тоог хамгийн ойрын бүхэл тоо хүртэл тоймло.

Жишээлбэл, дээр:

* 0 дахь өдөр: Нийт 1 компьютер халдвар авсан байна.
* 1 дэх өдөр: Нийт 2 компьютер халдвар авсан байна.
* 2 дахь өдөр: Нийт 4 компьютер халдвар авсан байна.
* 3 дахь өдөр: Нийт 8 компьютер халдвар авсан байна. Дараа нь нөхөөсийг хэрэглэнэ: 8 халдвартай \* 20% = 1.6 нөхөөстэй. 1.6-аас 2 хүртэлх тойрог. 8 компьютер халдвар авсан - 2 засвар хийсэн = 3 дахь өдрийн дараа халдвар авсан нийт 6 компьютер.

Өгөгдсөн хоног өнгөрсний дараа нийт халдвар авсан компьютерийн тоог буцаана уу.
