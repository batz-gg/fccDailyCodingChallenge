# Зураг хайх

2001 оны 11-р сарын 4-нд Google зураг хайлтаа эхлүүлж, хүмүүст хайлтын үг ашиглан зураг олох боломжийг олгосон. Энэ сорилтод та зураг хайлтыг дуурайх болно. Зургийн нэрс болон хайлтын нэр томъёоны массив өгөгдсөн бол хайлтын нэр томъёог агуулсан зургийн нэрсийн массивыг буцаана уу. Хайлтын үгтэй таарахдаа тохиолдлыг үл тоомсорло. Зургийг оролтын массив дээр харагдах дарааллаар нь буцаана уу.
